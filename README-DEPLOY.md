# 🚀 Deploy Limpo - Sem Dependências

Esta pasta contém APENAS os arquivos necessários para deploy estático.
Use esta pasta para fazer deploy - ela não tem nenhum arquivo que cause problemas de dependências.

## 📁 Conteúdo:
- `Home.html` - Página inicial
- `Skills.html` - Página de habilidades
- `Projects.html` - Página de projetos
- `styles.css` - Estilos com animações
- `script.js` - JavaScript
- `imagem/` - Imagens
- `.nojekyll` - Para GitHub Pages
- `netlify.toml` - Para Netlify
- `vercel.json` - Para Vercel
- `_config.yml` - Configuração geral

## 🚀 Como usar:
1. Faça upload de TODOS os arquivos desta pasta
2. Configure `Home.html` como página inicial
3. Deploy automático - sem instalação de dependências!

## ✅ Garantido:
- Sem `package.json`
- Sem `vite.config.js`
- Sem arquivos `.jsx`
- Sem pastas `node_modules`
- Apenas HTML/CSS/JS puro
